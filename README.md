🍕 Pizza Byte – Full Stack Pizza Ordering Application

📌 Project Overview

Pizza Byte is a full-stack pizza ordering web application developed using the MERN Stack. The application allows users to register, log in, browse pizza varieties, build custom pizzas, order ready-made pizzas, make online payments through Razorpay, and track their order status. Administrators can manage inventory, monitor orders, update delivery status, and receive low-stock alerts.

🎯 Objectives

* Develop a complete pizza ordering platform using modern web technologies.
* Implement secure user authentication and authorization.
* Allow users to customize pizzas based on their preferences.
* Integrate Razorpay payment gateway in test mode.
* Build an inventory management system for administrators.
* Track order status from placement to completion.
* Automatically update stock after each order.
* Notify administrators when inventory falls below threshold levels.
* Provide a responsive and user-friendly interface.
